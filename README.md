# Fortlev Clone

Projeto de estudo que refiz o site da Fortlev Solar.com, feito para praticar 
conceitos de HTML e CSS aprendidos em aula.

##  Objetivo
Aplicar na prática temas como flexbox, posicionamento, tabelas, formulários 
e animações CSS.

## Tecnologias
- HTML5
- CSS3
